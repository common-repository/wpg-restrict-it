Restrict content in posts and pages with this plugin and display only when a user leaves comment.

== Description ==

This plugin hide content in posts and pages and display only when a user leaves comment. If page/post dont have a comment from user or its got deleated by admin, content will be not displayed. 

This plugin uses cookie for that so this plugin will work when user allow website to set cookie.

Restrict it provides you a shortcode [wpgrestrict][/wpgrestrict], wrap your content with this shortcode and you are done.


In next version, I am going to to add restrict content by logged in user. Please let me know if you need any other way to restrict content.




== Installation ==

Installation is easy...


1. Upload `wpg-restrict-it` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
1. Wrap your content with `[wpgrestrict][/wpgrestrict]` in post editor
